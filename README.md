# E-Commerce Sales Analysis using Python | KPI Analysis & Business Insights

## Project Description

An end-to-end data analysis project built using Python and Jupyter Notebook, focused on extracting key performance indicators (KPIs) and generating business insights from an e-commerce dataset (Sample Superstore). The project analyzes sales trends, profitability, customer behavior, and product performance.

---

## Project Objectives

Analyze overall sales and profit performance  

Identify peak and low-performing months  

Evaluate category and sub-category performance  

Understand customer segment contribution  

Analyze sales-to-profit efficiency  

Support business decision-making using insights  

---

## Key KPIs

Total Sales: $2,297,201.68  

Total Profit: $286,397.02  

Total Orders: 9994  

Average Sales per Order: $229.9  

Profit Margin: 12.47%  

---

## Key Insights

Highest sales observed in November  

Lowest sales observed in February  

Maximum profit recorded in December  

Technology category contributes highest sales and profit  

Furniture category shows low profitability  

Tables is a loss-making sub-category  

Consumer segment leads in both sales and profit  

Home Office segment shows better efficiency  

Sales peak during Q4 (festive season)
